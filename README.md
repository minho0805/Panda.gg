# Panda.gg 🎮

**Panda.gg**는 League of Legends(롤) 전적을 검색할 수 있는 웹사이트입니다.  
[Riot Games API](https://developer.riotgames.com/)를 활용하여 소환사의 최근 게임 정보와 기본 통계를 조회할 수 있습니다.

## 🚀 주요 기능

- 소환사 이름으로 검색
- 최근 전적 정보 출력
- 챔피언, 킬/데스/어시(KDA), 승/패 여부 표시
- Riot API 연동
- Flask로 구현된 간단한 백엔드 서버

---

## 🛠️ 기술 스택

- Python 3.x
- Flask
- Riot Games API
- HTML/CSS (Jinja 템플릿)
